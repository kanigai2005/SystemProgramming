# 🖥️ System Programming in C

> A collection of System Programming projects implemented in **C** to understand Linux internals, process management, threading, scheduling, interrupt handling, debugging, and system-level programming concepts.

---

## 📚 About

This repository contains various mini-projects and implementations created while learning **System Programming**.

The goal of this repository is to gain hands-on experience with concepts such as:

- Process Management
- POSIX Threads
- CPU Scheduling
- Interrupt Handling
- Tasklets & Workqueues
- Shell Implementation
- Memory Management Basics
- System Monitoring
- Debugging Techniques
- Linux Kernel Programming Concepts

Each program focuses on understanding one important system programming concept through implementation.

---

## 🛠️ Tech Stack

- **Language:** C
- **Platform:** Linux
- **Compiler:** GCC
- **Debugger:** GDB
- **Libraries:**
  - pthread
  - stdio
  - stdlib
  - unistd
  - signal
  - syscalls
  - Linux Kernel APIs (where applicable)

# 📂 Repository Structure

| File | Description |
|------|-------------|
| `Db_Manager.c` | Simple key-value database manager implementation |
| `InterruptTasklet.c` | Demonstrates interrupt handling using tasklets |
| `InterruptWorkqueue.c` | Interrupt handling using Linux workqueues |
| `Manager_Worker_Sched.c` | Manager-Worker scheduling implementation |
| `MyShell.c` | Custom Linux shell supporting command execution |
| `System_Monitor.c` | Displays system resource usage and monitoring |
| `ThreadCreation.c` | POSIX thread creation examples |
| `ThreadSchedule.c` | Thread scheduling and priorities |
| `gdb.c` | Debugging practice program for GDB |
| `interrupt.c` | Basic interrupt handling concepts |
